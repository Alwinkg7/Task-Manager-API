Task Manager with User Registration

The Task Manager is a comprehensive web-based application designed to improve task management by incorporating user registration and authentication. This feature-rich application enhances productivity, facilitates collaboration, and ensures data security by associating tasks with individual users.
________________________________________
Key Features:
1.	User Registration & Authentication:
o	Users can create accounts and log in securely.
o	Authentication ensures that tasks are user-specific and private.
2.	Task Management:
o	Task Creation: Users can create tasks with details such as title, description, and deadline.
o	Task Viewing: View and manage tasks in a clean and structured interface.
3.	Task Assignment:
o	Assign tasks to registered users for efficient team collaboration.
4.	Task Status Updates:
o	Mark tasks as completed or update their details.
________________________________________
Technologies Used:
•	Frontend:
o	React.js: For a dynamic and responsive user interface.
o	CSS: To design an appealing and user-friendly layout.
•	Backend:
o	FastAPI: A modern, high-performance Python web framework for API management.
o	Fetch API: For seamless communication between the frontend and backend.
•	Database:
o	SQLite (or an equivalent relational database) to store user and task data.
•	Authentication:
o	JSON Web Tokens (JWT) for secure user authentication and session management.
________________________________________
Benefits:
1.	Personalized Experience:
o	Each user has their own space to manage tasks, ensuring privacy and organization.
2.	Improved Collaboration:
o	Assigning tasks to specific users enhances teamwork and accountability.

4.	Scalability:
o	The architecture allows for easy addition of features like team-based task boards.
5.	User-Friendly:
